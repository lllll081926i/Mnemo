# Mnemo v0.1.1-preview.1

Windows 预览构建（prerelease），用于验证多网盘 OAuth、能力路由与近期稳定性修复。

## 本预览包含

- 多账号应用内 OAuth 基础设施与聚合登录
- 网盘侧边栏与文件能力路由
- 多网盘目录 / 预览 / 分享 / 文件操作补全
- Google Drive 文件操作补全
- aria2 启动竞态与 Node 25 相关修复
- 窗口标题栏拖动区域调整
- 阿里登录 webview 生命周期清理
- 移除未使用依赖 `markdown-it`
- 项目审查文档 `docs/PROJECT_REVIEW.md`

## 注意

- 本版本为 **预览版**，不作为正式 latest release
- 国际盘（尤其 Google Drive / GoFile）能力仍在补齐中
- OAuth 需仓库 Secrets 配置齐全方可完整登录对应网盘

## 安装

下载本 Release 中的 Windows `.exe` 安装包进行试用，问题请提 Issue。
