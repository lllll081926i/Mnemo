# BoxPlayer v5.0.2

> 发布主题：新增光鸭云盘、多网盘工具完善、115 播放与限流修复、AI 工具调用补齐、GitHub Secrets 自动发布链路。

## 发布摘要

本次版本正式新增光鸭云盘接入，并对多网盘工具、视频播放、媒体整理、后台扫描和 AI Agent 工具调用做了一轮集中完善。重点修复了第三方网盘播放误用阿里云盘转码逻辑、115 非原画清晰度播放失败、115 后台扫描限流 toast 刷屏，以及多网盘扫描/整理工具能力边界不一致的问题。

同时，本版本引入 GitHub Secrets 发布方案：本地使用 `.env.local`，CI 发布使用 GitHub Secrets，构建时自动生成 `src/secrets.generated.ts`，避免公开仓库提交真实 Client ID、Client Secret 和私有 API 地址。

## 新增能力

- 新增光鸭云盘作为新网盘能力，支持登录、目录浏览、文件信息、下载、上传和视频播放等基础链路。
- 光鸭云盘能力接入 AI tool call，AI 可以感知并调用光鸭相关工具能力。
- 新增书籍库后台自动扫描，可在设置中按账号控制音乐、视频、书籍三类自动扫描。
- 媒体库 AI 刮削改为使用 BoxPlayer Cloud 专用接口，减少本地 BYOK 模型差异导致的刮削不稳定。
- 发布流程支持 tag 自动创建 GitHub pre-release，并自动把 `docs/releases/<tag>.md` 同步为 GitHub Release Notes。
- 新增 Promote Release workflow，可在人工审批后把 pre-release 转为正式 release。

## 多网盘工具

- 空目录扫描、重复文件扫描、大文件扫描继续向多网盘能力统一，减少阿里云盘专属限制。
- 移动整理、媒体整理和插件工具补齐网盘能力判断，避免跨网盘或不支持能力时误执行。
- AI Agent 补齐多网盘工具调用，包括扫描空目录、扫描大文件、扫描重复文件、整理文件、媒体整理和安全删除等。
- 删除语义收紧：AI `deleteFiles` 只对明确支持回收站/安全删除的网盘执行，不再对所有网盘承诺“移入回收站”。

## 播放修复

- 修复第三方网盘视频播放可能误走阿里云盘转码接口的问题。
- 修复 115 网盘默认清晰度设置为 1080P 时仍进入 4K 的问题。
- 修复 115 非原画清晰度在内置网页播放器中提示“不支持当前媒体类型”的问题，补齐 HLS 类型识别。
- 123、115、阿里云盘转码结果补齐视频类型信息，播放器可以更稳定选择播放内核。

## 115 网盘优化

- 为 115 文件列表请求增加节流队列，降低短时间连续请求触发访问上限的概率。
- 后台静默扫描遇到 115 限流时不再连续 toast 刷屏，改为日志记录或限频提示。
- 115 访问上限触发后增加短时冷却，避免后台扫描继续打爆接口。

## Secrets 与发布

- 新增 `.env.example` 和 `src/secrets.example.ts`，明确本地与 CI 需要配置的私有变量。
- 新增 `scripts/generate-secrets.mjs`，本地读取 `.env.local`，CI 读取 GitHub Secrets。
- `pnpm run dev`、`pnpm run build`、`pnpm run test` 前会自动生成 `src/secrets.generated.ts`。
- `src/secrets.generated.ts` 和 `.env.local` 均被忽略，不会提交真实密钥。
- BoxPlayer Cloud API 地址 `BOXPLAYER_AI_API_URL` 已迁移到 Secrets，不再硬编码在运行时代码中。

## 修复问题

- 修复光鸭云盘接入过程中视频播放失败问题。
- 修复 115 网盘频繁提示“已达到当前访问上限”的刷屏体验。
- 修复 115 网盘在 DevTools 中不明显展示 API 调用时，后台扫描实际连续触发请求的问题。
- 修复多网盘工具页部分入口和能力边界不清晰的问题。
- 修复未匹配媒体在未启用内部 AI 刮削时可能没有正确入库的问题。
- 修复本地/CI 构建时 Client ID、Client Secret 访问方式不一致导致发布包无法获取登录二维码的问题。

## 升级建议

- 使用光鸭云盘、115 网盘、123 网盘、多网盘扫描工具和 AI 整理能力的用户建议升级。
- 首次使用本版本发布流程前，需要在 GitHub 仓库配置 Release 所需 Secrets。
- 删除、整理、批量扫描等操作建议先在小目录验证，再对大目录批量执行。

## 已知限制

- 不同网盘的删除和回收站语义不一致，本版本优先保证安全：能力不明确时会跳过或提示，不冒充回收站。
- 部分网盘的离线下载、上传、分享能力可能受官方开放平台权限、会员状态或账号风控影响。
- macOS 包已纳入自动 Release matrix，当前默认生成未公证的 dmg/zip 包；如需正式签名和公证，需要额外配置 Apple Developer 证书相关 Secrets。
