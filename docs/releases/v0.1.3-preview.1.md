# Mnemo v0.1.3-preview.1

全量代码库审查与死代码/残留功能深度治理后的预览构建（prerelease）。

## 本次更新内容

### 死文件与无用资源清理
- **移除废弃 HTML 入口**：删除 `public/` 下遗留的 `main.html`、`main2.html` 及 `mainLyric.html`。
- **清理悬空 Worker / WASM 脚本**：移除早期阿里云盘计算哈希的 `sha1filework.js`、`wasm.wasm` 及 `wasm_exec.js`。
- **瘦身无用媒体与图标资源**：删除未引用的上传提示音 `upload_finished.mp3`、打赏与促销二维码图片，以及已下架云盘 Provider 的 SVG 图标。
- **清理编译中间产物**：清除 `electron/main/` 源码树中手动编译遗留的 `.d.ts` / `.js` 文件以及根目录 `nano-staged.mjs`。

### 死代码与孤儿组件剥离
- **剥离废弃类与接口存根**：彻底清理 `AliDirFileList` 废弃类与 `AliShareList` 模块及测试。
- **移除无入口的孤儿组件**：删除 `SearchPanModal.vue`（阿里云高级搜索）、`PageVideoXBT.vue`（视频雪碧图）、`PostModal.vue`（系统公告）、`M3U8DownloadModal.vue`（M3U8下载）及对应关联代码。
- **清理废弃工具导出**：清理 `utils.ts`、`shareurl.ts`、`electronhelper.ts` 中无引用的历史工具导出。

### Bug 修复与残留逻辑清洗
- **修复生产环境二级窗口 URL 报错**：修复 `electron/main/core/window.ts` 生产打包模式下加载不存在的 `dist/main2.html` 导致 404 的问题，统一指向 Vite 打包产物 `dist/index.html`。
- **工具函数规范重构**：将拼写错误的 `mosehelper.ts` 重构更名为 `mousehelper.ts`，导出语义清晰的 `isLeftClick` 函数。
- **播放器关闭文案修复**：修正关闭 MPV 播放器窗口时的误导提示文案（改为“播放完毕，已自动关闭播放器”）。
- **清理阿里云双盘硬编码残余**：彻底替换菜单与工具栏中创建分享或回退主目录时硬编码的 `'resource_root'` / `'backup_root'` 节点。

## 质量验证

- 单元测试全量通行（62 个测试套件，304 个测试用例）。
- Vue 模板与 Electron 主进程 TypeScript 类型检查全量通过。
