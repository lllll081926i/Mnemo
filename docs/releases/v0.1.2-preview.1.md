# Mnemo v0.1.2-preview.1

Windows 预览构建（prerelease），包含多云存储架构的全面死代码清理、冗余功能剪除及关键功能优化。

## 本次更新内容

### 死代码与冗余清理
- **移除旧版阿里云盘推销与未用逻辑**：删除 `alipanPromotionRules` 推广规则及对应测试。
- **清理已废弃上传与哈希逻辑**：彻底清理原阿里云盘特定上传栈（`upload.ts` / `uploaddisk.ts` / `uploadhashpool.ts` / `uploadmem.ts` / `sha1workerpool.ts`）。
- **清理全局与目录树遍历冗余**：移除无调用的全盘枚举（`dirlist.ts`）与目录树 walk 模块（`filewalk.ts` / `CopyFileTreeModal`）。
- **裁剪非目标产品功能**：删除 `CreatNewFileModal`（新建文本文件）、`ArchiveModal`（压缩包在线解压），短路分享导入及收藏夹清空等恒为 false 的能力位与死菜单。
- **清理 AI 与辅助整理模块**：彻底删除 `src/utils/drive-tools/`（`organize` / `mediaOrganize` / `directLinks`）以及网盘界面相关按钮和样式残留。

### 功能与稳定性优化
- **提供商目录与回收站统一**：新建 `src/utils/providerDirList.ts`，为 PikPak / Google Drive / OneDrive / Dropbox / GoFile / WebDAV / S3 提供统一的目录与回收站列表接口。
- **回收站清空修复**：`topTrashDeleteAll` 改用 `listProviderTrashItems` 异步查询与批量删除，修复 PikPak / Google Drive 回收站清空逻辑。
- **Aria2 文件夹下载修复**：文件夹下载展开改用 `listProviderDirChildren`，全面覆盖 7 大活跃提供商。
- **图片预览与代理路由**：图片预览统一通过应用代理服务器路由，移除已退役的阿里云盘直链回落。
- **OneDrive 简单上传阈值对齐**：简单上传阈值调整为 4MB（与 Graph API 指南对齐），大于 4MB 统一使用分片会话上传。

## 质量验证

- 单元测试与集成测试全量通过（62 个测试套件，257 个测试用例）。
- Vue 模板与 Electron 主进程 TypeScript 类型检查全量通过。
